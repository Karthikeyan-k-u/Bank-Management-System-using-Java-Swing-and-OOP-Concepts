# Bank Management System (Java Swing)

## 📌 Overview

This is a GUI-based Bank Management System developed using Java Swing. It allows users to create accounts, deposit money, and withdraw funds with proper validation and exception handling.

---

## 🚀 Features

* GUI-based application using Java Swing
* Supports Savings and Current accounts
* Deposit and withdrawal functionality
* Custom exception handling
* Overdraft facility for current accounts
* Displays account details dynamically

---

## 🛠️ Technologies Used

* Java
* Swing (GUI)
* Object-Oriented Programming (OOP)
* Exception Handling
* Collections (ArrayList)

---

## 📂 Project Structure

* `Account` → Abstract class
* `SavingsAccount`, `CurrentAccount` → Child classes
* `Bank` → Manages accounts
* `MainFrame` → GUI class
* `Fulllength3.java` → Main class

---

## ▶️ How to Run

1. Open Command Prompt / Terminal
2. Compile the program:

   ```bash
   javac Fulllength3.java
   ```
3. Run the program:

   ```bash
   java Fulllength3
   ```

---

## 📊 How It Works

* User enters name, amount, and account type
* Clicks **Create Account** to create account
* Can deposit or withdraw money
* System validates input and handles errors
* Results are displayed in the output area
